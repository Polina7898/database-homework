# Домашнее задание по нормализации БД

**Кузнецова Полина**

## Что здесь

Это мое домашнее задание по созданию и нормализации базы данных. Взяла данные о клиентах и транзакциях, привела их к 3НФ.

## Файлы

- `Кузнецова_Полина_ДЗ1.ipynb` - jupyter notebook с решением
- `database_diagram.dbml` - схема БД для dbdiagram.io
- `normalization_report.md` - описание процесса нормализации
- `01_create_tables.sql` - скрипт создания таблиц
- `02_insert_data.sql` - скрипт загрузки данных
- папка `data/` - CSV файлы с данными

## Как запустить

1. Открыть notebook в Jupyter
2. Запустить все ячейки
3. Для создания БД в DBeaver выполнить SQL скрипты

## Структура БД

Получилось 4 таблицы:
- locations - локации
- products - товары  
- customers - клиенты
- transactions - транзакции

Подробнее в файле normalization_report.md
